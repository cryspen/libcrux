# KMAC

![verified](../../../.assets/verified-brightgreen.svg)

This crate implements a keyed MAC (KMAC) with SHA-3 as defined in [NIST SP 800-185](https://csrc.nist.gov/pubs/sp/800/185/final). It provides KMAC_128 and KMAC_256.

## Verification

The code in this crate is verified for runtime safety using F*.
